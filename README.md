# PROYECTO DE PREDICCIÓN DEL RENDIMIENTO ESTUDIANTIL UNIVERSITARIO MEDIANTE TÉCNICAS DE APRENDIZAJE AUTOMÁTICO

Este proyecto se desarrolla para predecir el rendimiento académico de un estudiante universitario a través de su nota final del curso, consta de 6607 registros de una universidad de EEUU, almacenadas en este mismo repo con la extensión csv. 

## 📌 Carga de datos
Lo primero que se debe realizar para correr el proyecto es descargar la data de nombre rendimiento_estudiantil.csv

## Abrir el Colab
Lo segundo consiste en dirigirse al archivo de extensión .ipynb de modo que se abrirá un documento en Google Colaboratory, donde ahí se debe cargar el archivo descargado previamente. Luego, debemos correr cada celda o todas en su defecto. Podremos acceder a información de exploración de datos, preprocesamiento hasta finalizar en el modelo propuesto que es el LinearRegression. 


### 🚀 Tecnología empleada

- Python (Librerías Pandas, Matplotlib y Seaborn)
- Google Colaboratoy
